# Instruções para rodar o backend

No diretório do projeto, você pode executar:

### Passo1: `npm intall`

### Passo2: Renomeie o .env.example para .env e coloque esses valores para funcionar

PORT=3001 \
DB_USER=samuel \
DB_PASS=4p0Je14peEXxXUR6 \
DB_NAME=revert

### Passo3: `npm start`

Executa o aplicativo no modo de desenvolvimento.\
Abra [http://localhost:3001/api](http://localhost:3001/api) para visualizá-lo em seu navegador.
